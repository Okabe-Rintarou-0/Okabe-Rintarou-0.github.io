形式化验证之 TLA+ 入门

2025/8/6 TLA+

安装

brew install --cask tla+-toolbox

案例——判断死锁

------------------------------ MODULE deadlock ------------------------------

EXTENDS Integers
VARIABLES locks, locksA, locksB

Init == 
    /\ locks = {}
    /\ locksA = {}
    /\ locksB = {}

A_REQUIRE_LOCK_1 ==
    /\ ~(1 \in locks)
    /\ locks' = locks \cup {1}
    /\ locksA' = locksA \cup {1}
    /\ locksB' = locksB

A_REQUIRE_LOCK_2 ==
    /\ 1 \in locksA
    /\ ~(2 \in locks)
    /\ locks' = locks \cup {2}
    /\ locksA' = locksA \cup {2}
    /\ locksB' = locksB
    
B_REQUIRE_LOCK_1 ==
    /\ 2 \in locksB
    /\ ~(1 \in locks)
    /\ locks' = locks \cup {1}
    /\ locksB' = locksB \cup {1}
    /\ locksA' = locksA

B_REQUIRE_LOCK_2 ==
    /\ ~(2 \in locks)
    /\ locks' = locks \cup {2}
    /\ locksB' = locksB \cup {2}
    /\ locksA' = locksA

Next == A_REQUIRE_LOCK_1
    \/ A_REQUIRE_LOCK_2
    \/ A_REQUIRE_LOCK_1
    \/ A_REQUIRE_LOCK_2
    
Spec == Init /\ [][Next]_<<locks, locksA, locksB>>
